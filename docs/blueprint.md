# **App Name**: ScholarSage

## Core Features:

- Profile Summarization: Summarize faculty profiles using an AI tool to quickly grasp research interests and expertise.
- Paper Summarization: Summarize research papers using an AI tool, extracting key findings and contributions for quick understanding.
- Faculty Directory: Display a searchable directory of faculty members with key information such as name, department, and contact details.
- News Feed: Aggregate university news and announcements in a central location, providing up-to-date information for faculty and students.
- Contact Form: Enable users to send messages through a contact form.

## Style Guidelines:

- Primary color: HSL(220, 50%, 60%) - A moderately saturated blue that reflects trustworthiness and knowledge. (#4882C2)
- Background color: HSL(220, 20%, 95%) - A very light, desaturated blue providing a clean and calm backdrop. (#E6EAF2)
- Accent color: HSL(190, 50%, 40%) - A contrasting cyan that highlights interactive elements. (#34A7A7)
- Body and headline font: 'Inter', sans-serif, for a clean, modern, objective look throughout the app.
- Use minimalist icons from Lucide to represent various sections and actions.
- Employ a clean, card-based layout to organize information effectively.
- Subtle animations for transitions and loading states to improve user experience.
